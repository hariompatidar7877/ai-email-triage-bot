# ✨ Project Overview

📬 **AI Email Triage Bot** is an **Agentic AI automation system** that connects Gmail with a Large Language Model to automatically classify incoming emails into:

```
Spam
Urgent
Inquiry
```

Instead of manually sorting emails, the AI understands context and categorizes them intelligently ⚡

---

# 🎯 Features

✅ Gmail API integration
✅ OAuth secure authentication
✅ Groq LLM classification engine
✅ Automatic MIME email parsing
✅ Token-efficient inference pipeline
✅ Modular Python architecture
✅ Environment variable security (.env)
✅ Debug-ready logging system

---

# 🧠 How It Works

```
Incoming Gmail Messages
        ↓
Gmail API Fetch
        ↓
MIME Email Parser
        ↓
Groq LLM Reasoning
        ↓
Spam / Urgent / Inquiry Output
```

# ⚙️ Installation

Clone repository

```
git clone https://github.com/YOUR_USERNAME/ai-email-triage-bot.git
cd ai-email-triage-bot
```

Install dependencies

```
pip install -r requirements.txt
```

Create `.env` file

```
GROQ_API_KEY=your_api_key_here
```

Add Gmail OAuth credentials

```
credentials.json
```

Run application

```
python app.py
```
